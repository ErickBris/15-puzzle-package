Getting Started

To start with the project you have to perform some simple tasks.
1. Import the project into your eclipse IDE.
2. Modify admob ads into the project.
3. Modify Goolge analytics (to track how your app is doing).
4. Changing the basics of project like app name, package name.
5. Reskinning the project with changing the graphic content present in the project.
6. Deploy in the play store.
Yep thats it . These are the only step you need to worry about . Each step is explained in detail given below.

Importing code into Eclipse

To import the code into your eclipse do the following steps
  1. Click on File
  2. Select New Project
  3. Select Android Project
In the dialog, choose the option "Existing Project" and put the path to the  project root path.

Eg - you have put code in C drive then put the path as Local Disk C
In the following list you will be shown the list of projects that you can select Then select Fifteenpuzzle.

Adding Admob ads

Note - This section is only for users using Google admob ads. If you dont want to use admob then skip this step. However you will be getting some errors due to missing library files. These errors will be removed by removing Adrequest and Adview varaibles.(Just delete the lines causing errors)

Another folder for google play services are provided with the code. Just copy the code and paste into the same place where you imported your eclipse project. Normally it is C:\Users\<-- your computer user name-->\workspace

To import the services do the following steps
  1. Right Click on your project in eclipse project explorer
  2. Select Properties
  3. In the dilaog select android
  4. In the library section select add then select the google play games library
  5. select Ok.
Note - Do change the admob id that you get from admob with the one already given in the code by following steps

Open the code in eclipse then go to res--> values ---> strings then you will see a line like given below
 <string name="interstitial_id">a15305056466330</string>
Delete a15305056466330 and add your admob id over here it should look like this
 <string name="interstitial_id">your_Id</string>

 

 

Integrating Google Analytics

To start Using Google Analytics you need to have the following pre-requisites
  •  A google analytics Acccount (duhh)
  •  A tracker Id (Given to you in google analytics).
Open the code in eclipse then go to res--> values --->analytics.xml
You will see a string like this give below
 <string name="ga_trackingId">UA-47805140-1</string>
Change it to your new tracking id
 <string name="ga_trackingId">YOUR-TRACKING-ID</string>
 
To obtain Tracking Id
check out the link below 
https://support.google.com/analytics/answer/1032385?hl=en

Removing Errors(If any)

If you recieving any errors most likely R cannot be resolved to a variable or android cannot be resolved to any type then do the folllowing
  1. Right Click on your folder in package explorer in eclipse
  2. Slect properties
  3. Then select Android
  4. Then in the android build target folder select the latest sdk you have.
  5. Select OK.

In eclipse Go to

then you will see a line like this

Changing Package Name

To change package name 
Right click on your project in eclipse, scroll down to Android Tools, and then click on Rename Application Package.

Enter your name and hit Ok

Changing Text in Game

To change the text present in game do the following steps

In eclipse Go to 
 res->values->strings

there you will see lines like these

    <string name="dialogTitle">Resume Game</string>

    <string name="yes">Yes</string>

    <string name="no">No</string>

and many other lines

 
All the code is properly formatted with instruction about which screen these strings will show
To change them to your values just change the content in it.
 
Example - 
Before changing - 
<string name="dialogTitle">Resume Game</string>

After changing - 
<string name="dialogTitle">Your Text Here</string>

    <color name="grey">#cdcccc</color>

    <color name="light_green">#d1feca</color>

    <color name="red">#e93030</color>

    <color name="blue">#0207eb</color>

Changing Graphics

To change graphic images Go to your project folder -> res
you will see folders like
drawable-mdpi
drawable-ldpi
drawable-hdpi
drawable-xhdpi
drawable-xxhdpi
drawable-large-hdpi
drabwale-xlarge-mdpi

Thes folders contain graphics that are used in the game for different types of devices
Just change the images to your images and DO NOT RENAME YOUR NEW IMAGE TO A DIFFERENT NAME


Eg - if your are changing an image theme_1.png
then replace it with another image with the name as theme_1.png

Devices with their notations 
ldpi - devices with resolution 240x320.
mdpi - devices with resolution 320x480.
hdpi - devices with resolution 480x800.
xhdpi - devices with resolution 720x1280.
xxhdpi - deives with resolution 1080x1920.
large-hdpi - for tablets like nexus 7.
xlarge-mdpi - for tablets like nexus 10.
